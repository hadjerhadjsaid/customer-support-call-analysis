# Customer Call Analysis with NLP

This project applies speech recognition and NLP techniques to analyze customer support calls.

## Features
- Speech-to-text conversion
- Audio quality verification
- Sentiment analysis using VADER
- Named entity recognition
- Text similarity detection

## Technologies
Python, SpeechRecognition, NLTK, spaCy, Pandas
Executed and validated on the DataCamp platform.
